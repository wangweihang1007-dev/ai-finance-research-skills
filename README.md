# AI Finance Research Skills

Reusable, model-agnostic instructions for AI coding agents that work on finance empirical research projects.

## Included Skills

- `skills/finance-empirical-layout/`: a finance empirical project organization standard for Gemini, Claude, Codex, ChatGPT, and other coding agents.
- `skills/finance-regression-table-export/`: export finance regression results into thesis-ready Word three-line tables.

## How To Use

Copy the relevant `SKILL.md` content into an AI agent's custom instruction, project instruction, or skill system. The standard requires agents to inspect or create a clear project directory before writing code, downloading data, or generating CSVs, figures, regression tables, logs, and reports.
